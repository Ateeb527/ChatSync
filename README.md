# ChatSync

Real-time chat application built using the MERN stack and Socket.io for instant messaging and live communication.

## Features
- Real-time messaging
- JWT authentication
- Secure password encryption with bcrypt
- Live message updates
- User search
- Responsive UI
- Redux persistence

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Socket.io
- Redux
- Tailwind CSS

## Challenges Faced
- Managing real-time socket connections
- Handling authentication securely
- Optimizing API and message updates
- State persistence and synchronization
